**Промт для платформы VibeCoding (Replit) с рабочим проектом сервиса для российских производств**

---

### **Цель проекта**  
Создать адаптивный под все экраны веб-сервис, объединяющий российские производства и клиентов. Основные функции:  
- **Бесплатное размещение карточек компаний** (название, описание, фото, контакты, каталог продукции).  
- **Бесплатный поиск и публикация заказов** для клиентов.  
- **Платные тарифы для производств** (услуги продвижения, аналитики, премиум-размещения).  

---

### **Анализ конкурентов**  
1. **https://msk.metalloobrabotchiki.ru/orders**  
   - Плюсы: фильтры по региону, сроку, бюджету; система откликов; разделение на категории.  
   - Минусы: устаревший дизайн, низкая адаптивность под мобильные устройства.  

2. **https://vsyametalloobrabotka.ru/orders/all_category/**  
   - Плюсы: простой интерфейс для публикации заказов; интеграция с картами (геолокация).  
   - Минусы: отсутствие сортировки по популярности, слабая аналитика для компаний.  

3. **https://сделаемвроссии.рф/**  
   - Плюсы: рекламные баннеры для премиум-пользователей; продвинутая сортировка заказов.  
   - Минусы: перегруженный интерфейс, высокие цены на тарифы.  

**Идеи для реализации**:  
- Универсальные фильтры с возможностью сохранения шаблонов.  
- Геолокационный поиск с картой (Google Maps или Yandex Maps API).  
- Интеграция с Яндекс.Метрика и Google Analytics для аналитики.  

---

### **Технические требования**  
#### **Фронтенд**  
- **Адаптивный дизайн** с использованием:  
  - **React.js + Tailwind CSS** (гибкость, скорость разработки, современный стиль).  
  - **Next.js** для SSR (SEO-оптимизация и быстрая загрузка).  
- **Библиотеки**:  
  - `react-responsive` для медиа-запросов.  
  - `axios` для API-запросов.  
  - `leaflet` или `@react-google-maps/api` для карт.  

#### **Бэкенд**  
- **Node.js + Express.js** или **Django REST Framework**.  
- **База данных**: PostgreSQL (структурированные данные) + Redis (кеширование).  
- **Аутентификация**: JWT + OAuth2 (вход через соцсети).  

#### **Серверная структура**  
```sql
-- Таблица компаний  
CREATE TABLE companies (  
  id SERIAL PRIMARY KEY,  
  name VARCHAR(255) NOT NULL,  
  description TEXT,  
  logo_url TEXT,  
  region VARCHAR(100),  
  tariff_id INT REFERENCES tariffs(id),  
  created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP  
);  

-- Таблица заказов  
CREATE TABLE orders (  
  id SERIAL PRIMARY KEY,  
  title VARCHAR(255) NOT NULL,  
  description TEXT,  
  budget DECIMAL(10,2),  
  deadline DATE,  
  customer_id INT REFERENCES users(id),  
  status VARCHAR(50) DEFAULT 'active'  
);  

-- Таблица тарифов  
CREATE TABLE tariffs (  
  id SERIAL PRIMARY KEY,  
  name VARCHAR(50) NOT NULL,  
  price INT NOT NULL, -- в рублях  
  features JSONB NOT NULL -- например: ["top_position", "analytics", "banner_ads"]  
);  

-- Таблица платежей  
CREATE TABLE payments (  
  id SERIAL PRIMARY KEY,  
  company_id INT REFERENCES companies(id),  
  tariff_id INT REFERENCES tariffs(id),  
  amount INT NOT NULL,  
  payment_date TIMESTAMP DEFAULT CURRENT_TIMESTAMP  
);  
```

### **Функционал платформы**  
#### **Для компаний**  
1. **Бесплатный аккаунт**:  
   - Размещение карточки компании.  
   - Отклик на 5 заказов в месяц.  
2. **Платные тарифы**:  
   - **«Старт» (2000 ₽/месяц)**: неограниченные отклики, размещение в топе по региону.  
   - **«Про» (5000 ₽/месяц)**: «Старт» + аналитика по откликам, баннерная реклама на главной.  
   - **«Бизнес» (15000 ₽/месяц)**: «Про» + персональный менеджер, продвижение в поиске.  

#### **Для клиентов**  
- Бесплатная публикация заказов.  
- Фильтры: бюджет, срок, регион, категория (металлообработка, пищевое производство, текстиль и т.д.).  
- Уведомления о новых откликах.  

---

### **Дизайн**  
- **Цветовая схема**: корпоративные оттенки синего и серого (ассоциация с надежностью).  
- **Шрифты**: «Montserrat» для заголовков, «Roboto» для текста (Google Fonts).  
- **Компоненты**:  
  - **Главная страница**: поиск заказов, карточки компаний с тегами («топ», «новые»).  
  - **Страница компании**: вкладки («О компании», «Продукция», «Отзывы»).  
  - **Личный кабинет**: управление тарифами, статистика откликов.  

---

### **Рыночный анализ**  
- **Целевая аудитория**: малые и средние производства (80% рынка), ищущие клиентов.  
- **Уникальное предложение**:  
  - Бесплатный базовый функционал.  
  - Гибкие тарифы с фокусом на ROI (возврат на инвестиции).  
  - Поддержка российского производства (патриотический фактор).  
- **Монетизация**:  
  - 30% компаний перейдут на платные тарифы (по данным исследований).  
  - Интеграция с государственными программами (гранты для производств).
